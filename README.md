## Description
A beginner-friendly terminal-based File Explorer written in **C++17** using the `<filesystem>` library.  
This program can:

- List files and folders  
- Create and delete files or directories  
- Rename, copy, and move files  
- View file contents  
- Search files by name  
- Show file size, modification date, and permissions
